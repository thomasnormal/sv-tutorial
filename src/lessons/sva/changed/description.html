$changed(sig) returns true when a signal's sampled value differs from its value at the previous clock edge â€” a concise way to write sig !== $past(sig):

update |=> $changed(data)

The companion function $sampled(expr) returns the value of an expression as sampled in the preponed region (the settled value at the clock edge). It is most useful inside action blocks to print what the checker actually saw:

else $error("data unchanged: 0x%0h", $sampled(data));

Open toggle_check.sv and complete the property body: while update is high, data must change every cycle.

$changed uses four-state inequality (!==) so it is also true when a signal transitions to or from X. To ignore X transitions use data != $past(data) (two-state inequality) instead.
