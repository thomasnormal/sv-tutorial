SystemVerilog's logic type has four states: 0, 1, X (unknown), and Z (high-impedance). In simulation, X propagates through logic and can silently corrupt results.

$isunknown(expr) returns 1 if any bit of expr is X or Z, making it a precise detector for uninitialized registers and floating buses.

Open xcheck.sv and complete the property body: the design guarantee is that data must never contain X or Z after reset. Negate $isunknown to express this as a positive assertion.

$isunknown is a simulation-only function. Formal tools work in two-state (0/1) logic and X/Z do not exist. Use it to catch bus contention, missing reset, or undriven ports during simulation regression.
