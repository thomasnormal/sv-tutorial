Logical Equivalence Checking (LEC) proves that two circuit implementations produce identical outputs for every possible input — without running any simulation.

If the solver finds an input that makes the two designs disagree, it reports sat and outputs a counterexample (the distinguishing input values). If no such input exists, it reports unsat — the designs are equivalent for every possible input.

Boolean identities

De Morgan's laws and the mux expansion are the most common equivalences in RTL synthesis:

Name Original Equivalent
De Morgan (NAND)~(a & b)~a | ~b
De Morgan (NOR)~(a | b)~a & ~b
Mux expansionc ? a : b(c & a) | (~c & b)

Logic synthesis routinely applies these transformations — and LEC proves the result is identical to the RTL spec.
